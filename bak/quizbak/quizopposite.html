3.3 Quiz

There are 4 questions to be attempted in sequence and scored.

Question 1 - When 2 objects move in the opposite direction, will they ever meet? (show the diagram below)

Yes / No (Answer 1 - Yes)

Feedback 1 - If answer is correct, provide feedback. Sid will say. “Good job!” Then move to Question 2 for user to attempt.

If answer is incorrect, provide feedback. Sid will say, “Try again.” Then move to Question 1 for user to attempt again.

Question 2 - 2 objects start moving in the opposite direction at the same time. If they move at the same speed, where will they meet? Drag the “Meeting Point” to where you think they will meet. (show the diagram below with slider for “Meeting Point”)

Answer 2 - The Meeting Point will be exactly at the midpoint indicated in red in diagram below.

Feedback 2 - If answer is correct, Sid provides feedback: “Good job!” Then move to Question 3 for user to attempt. If answer is incorrect, Sid provides feedback: “If the objects are travelling at the same speed, it means they cover the same distance at any time. How would this affect their meeting point? Try again.” Then move to Question 2 for user to attempt again.

Question 3 - 2 objects start moving in the opposite direction at the same time. If they move at different speeds, where will they meet? Drag the “Meeting Point” to where you think they will meet. (show the diagram below with slider for “Meeting Point”)

Answer 3 - The Meeting Point can be anywhere on the red line indicated in diagram below, excluding Point A and midpoint.

Feedback 3 - If answer is correct, Sid provides feedback: “Good job!” Then move to Question 4 for user to attempt. If answer is incorrect, Sid provides feedback: “If the object on the right is travelling at a faster speed, it means it covers more distance compared to the object on the left. How would this affect their meeting point? Try again.” Then move to Question 3 for user to attempt again.

How would this affect their meeting point? Try again.” Then move to Question 4 for user to attempt again.
